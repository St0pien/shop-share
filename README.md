## Tech Stack

- **Framework:** [T3 Stack](https://create.t3.gg/)
  - Next.js
  - TypeScript
  - Tailwind CSS
  - TRPC
- **Authentication:** [Auth.js](https://authjs.dev/)
- **Database:** PostgreSQL
- **SQL query builder | ORM | Migrations**: [Drizzle](https://orm.drizzle.team/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.dev/)
